# Email Rewriting Tone Changer 

A smart web app that rewrites emails in different tones using Gemini API. Built for Prompt Engineering Project 2.

## Features

- Rewrite any email into tones like Professional, Friendly, Assertive, etc.
- Simple and clean web UI (React )
- Powered by OpenAI's GPT-3.5-turbo
- Real-time prompt engineering preview

## Tech Stack

- React (Vite)
- OpenAI API (`gemini`)
